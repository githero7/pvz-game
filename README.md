# Plants vs. Zombies - JavaScript Clone

Play online: [https://githero7.github.io/pvz-game/](https://githero7.github.io/pvz-game/)

## About

A pure JavaScript clone of Plants vs. Zombies. No frameworks, no runtime — just open in a browser.

⚠️ **Disclaimer**: This project is for educational purposes only. Plants vs. Zombies is a registered trademark of Electronic Arts / PopCap. All characters, artwork, audio, and lore belong to their respective owners.

## Quick Start

1. Clone the repo
   ```bash
   git clone git@github.com:githero7/pvz-game.git
   ```

2. Open `index.html` in your browser — that's it.

## Project Structure

```
.
├── index.html              # Main page
├── assets/
│   ├── css/style.css       # Stylesheet
│   ├── js/
│   │   ├── core.js         # Core game logic
│   │   └── levels.js       # Level data

```
